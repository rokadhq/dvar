# Support Policy

Community support is provided through GitHub issues and discussions on a best-effort basis.

Bug reports should include the Dvar version, Node.js version, policy with sensitive values removed, minimal reproduction, expected decision, actual decision, and reason code. Security reports must follow `SECURITY.md` and must not be filed publicly.

Enterprise implementation, agent security assessments, MCP hardening, private deployment, and support agreements are available from Rokad separately from the open-source runtime.
